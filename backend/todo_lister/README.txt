Pasos para probar este codigo:
1) En el directorio del proyecto ejecutar un comando "composer install".
2) Para ejecutar el programa, se debera ejecutar el comando "symfony serve -d" en el directorio del proyecto
3) Se debera hacer inicio de sesion usando el usuario:
  -Nombre de usuario: admin
  -Contraseña: admin
  
Tiempo estimado de cada tarea:
1) (35min) Crear estructura para proyecto Symfony
2) (10min) Crear entidad TODO
3) (1,5h) Crear interfaz para añadir elementos a la lista y visualizar el listado.
4) (30min) Posibilidad de marcar como realizada la tarea y diferenciar visualmente de las pendientes.
7) (1,5h) Añadir login y autenticar la aplicación. Crear un rol admin para gestionar todas las listas. Un rol usuario accederá únicamente a su lista.
8) (1,5h) Añadir para un usuario con rol admin la posibilidad de modificar el dueño de la lista, pudiendo un usuario tener varias listas de TODO.
9) (1h) (Opcional) Añadir creación de usuarios.
10) (45min) Un admin puede visualizar las listas TODO del resto de usuarios, pero no modificar sus elementos.
